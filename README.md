# aws-telegram-bot
My personal AWS deploy tools


### This project provider a CLI to AWS EC2 from a Telegram bot

### Install  
  ```sh
    npm install
  ```
### Configure  
Create the env file:  
```
touch .env
```
Add the variable **TELEGRAM_TOKEN=XXXXXXXXXXXXX**  
*Replace XXXXXXXXXXXXX with your bot token*

  **build**
